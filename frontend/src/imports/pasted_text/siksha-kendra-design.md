# Siksha Kendra - Complete Website & Dashboard Design Prompt

Design a modern, premium, mobile-first EdTech SaaS platform called "Siksha Kendra".

1. Super Admin
2. Vendor
3. Student
4. Public Visitor

The design must be production-ready, professional, scalable, and suitable for schools, coaching institutes, and educational organizations.
